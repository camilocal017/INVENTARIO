# **App Name**: Kitchen Command

## Core Features:

- Product Entry: Add new kitchen implements with details like name, description, and initial stock quantity.
- Stock Adjustment: Update stock levels for each item based on stock-in and stock-out events.
- Sales Recording: Record sales transactions, including items sold, quantities, and total sale amount.
- Sales Report Generation: Generate basic sales reports showing total sales, top-selling items, and sales trends over a specified period. As a tool, LLM decides if data should be filtered according to top sales.
- Search and Filter: Enable users to search and filter products by name or category to quickly find and manage inventory.

## Style Guidelines:

- Primary color: Steel blue (#4682B4) to evoke professionalism and reliability.
- Background color: Light gray (#F0F8FF), a desaturated tone of steel blue for a clean backdrop.
- Accent color: Soft gray-blue (#778899) to provide contrast and highlight key UI elements.
- Font: 'Inter', a grotesque-style sans-serif, for body and headlines to create a modern and machined look.
- Use simple, outline-style icons in a monochromatic steel blue to maintain a clean and professional look.
- Maintain a clean, structured layout with clear sectioning to ensure easy navigation and efficient data management.
